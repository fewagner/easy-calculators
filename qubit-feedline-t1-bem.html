Qubit–Feedline T₁ Estimator

Cmf and Cgnd are read from a BEM lookup table and linearly interpolated. T₁ is then calculated with input-output theory.

Method
Boundary-element method (BEM), Maxwell capacitance matrix
Substrate
Half silicon / half vacuum — εeff = (1 + εSi)/2 = 6.35
Feedline
Length 1.98 mm, centre-conductor width 10 μm, gap 5.5 μm each side
Boundary box
2 mm × 2 mm
Table grid
R ∈ {50, 100, 200} μm  ·  D ∈ {100, 200, 400} μm  ·  s ∈ {5, 10, 20} μm
Interpolation
Trilinear (linear in R, D, s independently)
⚠ Parameter combination is outside or near the edge of the BEM grid — result involves extrapolation and may be less accurate.

Formulas

QuantityFormula / Note
Cmf, Cgnd, Cself Linearly interpolated from BEM table in (R, D, s). Cmf = |Cfeedline,island|, Cgnd = |CGND,island|, Cself = Cisland,island (Maxwell matrix diagonal).
Angular frequency ω = 2πfq
T₁ (input-output theory) T₁ = CΣ / (2ω² Cmf² Z0)
Factor of 2 Two ports of an infinite matched feedline (power radiated both ways)
Validity ω Cmf Z0 ≪ 1 (weak coupling; typically ~10−4)
